### Dimension of Data
그림, 언어, 수... 우리 주변의 모든 것은 데이터이다.
데이터의 핵심 형태는 크게 5가지로 말할 수 있다.
- point: 점, 점 하나로만 정의되는 데이터 (ex [0,1]은 존재한다 또는 존재하지 않는다)

- scalar: 크기 있는 값(point는 값의 크기 아님) (ex 1,3,e, 자연상수... 실수, 정수, 허수 등의 수 체계 안에 속하는 값들)

- vector: scalar가 연달아 나열된, 한 차원 높아진 데이터 (하나의 벡터 안에 scalar가 n개 들어간다고 하면, a​∈R** n)

- matrix: vector에서 한 차원 높아진 것 (행렬 A에서 세로길이 n 가로길이 m인 즉4개짜리 벡터가 3개 있다고 한다면, A∈R** n * m)

- tensor: matrix에서 한 차원 높아진 것 (∈R**n * m * r)

### Linear Independent 벡터의 선형 독립
vector는 방향성이 있다(점을 가리키는 화살표로 이해하자)
쉽게 말해서, 벡터의 선형독립이란 두 벡터가 각자 얼마나 개성이 있느냐?를 말한다.

벡터 A와 B는 고유한 영역(직선)을 가진다. 2차원의 공간(R**2)안에 직선(R**1)이 있을때,

**두 직선으로 좌표평면 전체공간을 모두 덮을 수 있다면 A와 B는 선형독립이라고 할 수 있다.**
(덮을 수 없으면 종속이다)

이는 'av1​+bv2​=0을 만족하는는 a와 b가 없으면 선형독립이다!'로 나타낼 수 있다. (즉, av1​+bv2​=0 가 좌표평면 전체 영역 모두 덮을 수 있으면 선형독립이다)

엄밀히 말하면 선형독립은, “어떤 수를 곱해서 0이 되면 안 된다” ❌

“0으로 만드는 방법이 ‘전부 0을 곱하는 경우밖에 없으면’ 선형독립이다” ⭕

-> 두 벡터가 선형독립이라는 것은 영벡터를 만드는 선형결합이 자명한 해(모든 계수 0)밖에 없다는 것이다.

### Basis와 Span의 개념
n차원의 공간을 만들기 위해서는 벡터가 n개가 필요한데(R** 2 즉, 2차원을 만들기 위해서는 벡터가 2개가 필요하다)

n차원의 공간을 재구성할 수 있는 n개의 벡터 각각을 **Basis**라고 부른다.

이때, n개의 벡터가 R** n 공간(n차원의 공간)을 **span**할 수 있으면 선형독립이다.

### Matrix의 Rank의 의미 Matrix as Vector space, Rank
두 개의 벡터를 이어붙어보자

[2 1] 

[1 2]

[a,b]

∈R**2*2 -> 여기서 a 벡터와 b벡터는 각각 Basis Vector이며, 이 두 vector로 2차원 공간을 span할 수 있다. 

##이때, 우리는 이 행렬의 'Rank'를 2라고 부른다. 두 개의 벡터로 공간을 span할 수 있기 때문이다.##


### 선형독립의 조건
3 by 3 행렬의 예시를 보자, 

[1 0 0]

[0 1 0]

[0 0 1]

[a b c]

이때, 행렬의 Rank는 3이다. 3개의 벡터로 3차원 공간을 모두 채울 수 있기 때문이다. 

*그렇다면 행렬이 R** n * n일 때, 항상 Rank가 n인가?*
아니다.

[2 4]

[1 2]


이 경우 한 벡터가 다른 벡터의 배수이다. 즉, 같은 방향인 선형종속이다. 그러므로 선형독립이 아니다. 
이 경우 행렬의 Rank는 줄어들어 1이 된다. 

**그러므로 Rank가 n이기 위해서는**
R** n * n(행렬) 에서 vector들이 선형독립일 떄 Rank가 n이다.
서로 같은 수직선 위에 vector가 있으면 즉, 방향이 같으면 Rank가 떨어진다.
**그러므로 Rank가 보존될 조건은 선형독립이다**
(선형독립의 조건은 벡터에 상수를 곱해줬을 때 0이 되는 조합이 나오면 안된다!)

즉, 행렬의 rank는 열벡터들의 선형독립성에 의해 결정되며,
영벡터를 만드는 선형결합이 자명한 해만 갖는 경우,
그 독립성이 유지되어 rank는 보존된다.

### Summary
[선형독립] 영벡터를 만드는 선형결합이 자명한 해(모든 계수가 0인 경우)밖에 없을 때 벡터들은 선형독립이며,
[Rank] 행렬의 rank는 이러한 선형독립인 열(또는 행)벡터의 최대 개수로 정의되고,
[Rank 보존] 행렬의 열(또는 행)에 가해지는 변형이 이 선형독립성을 유지하는 경우, 즉 새로운 벡터가 기존 벡터들의 선형결합으로 표현되지 않을 때 행렬의 rank는 떨어지지 않고 보존된다

**rank는 선형독립 벡터의 개수이므로, 선형독립성이 유지되는 변형 하에서는 행렬의 rank가 보존된다.**
